Zaychess Support

Zaychess is a macOS chess app with local play, a built-in AI opponent (Serendipity engine), and direct peer-to-peer multiplayer by IP. You can save and load games as .chesslog files via the standard file picker.

Key features

  • Local games with a move list panel and fullscreen toggle.
  • Play vs computer AI (Serendipity engine) with adjustable difficulty.
  • Host or join multiplayer games by IP using direct peer-to-peer connections.
  • Save and load .chesslog files using the standard file picker.

System requirements

macOS 12.0 or later.

Networking

Multiplayer uses direct TCP connections between players. There are no servers in the middle.

Data and privacy

  • Data collection: No account, no tracking, no analytics; all data stored locally.
  • Network use: Only for optional multiplayer (direct TCP between players).
  • Privacy policy: jeremyzay.com/zaychess/privacy

Known issues

  • LAN multiplayer may require firewall permission on one or both machines.

Logs for support

  • Development builds: zaychess.log in the app directory.
  • Packaged app logs: check ~/Library/Logs/Zaychess/ (if present) or share a crash report from Console.app.
